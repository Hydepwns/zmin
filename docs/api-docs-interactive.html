zmin

High-performance JSON minifier API documentation

API Examples

Basic Minification

curl -X POST "https://api.zmin.dev/v1/minify" \
  -H "Content-Type: application/json" \
  -d '{"input": "{\"key\": \"value\"}"}'

# Response
{
  "output": "{\"key\":\"value\"}",
  "size_reduction": "12%",
  "processing_time": "0.001s"
}

Turbo Mode

curl -X POST "https://api.zmin.dev/v1/minify" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "{\"large\": \"dataset\"}",
    "mode": "turbo",
    "options": {
      "chunk_size": "1MB",
      "threads": 16
    }
  }'

GPU Acceleration

curl -X POST "https://api.zmin.dev/v1/minify" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "{\"massive\": \"dataset\"}",
    "mode": "turbo",
    "gpu": "cuda",
    "options": {
      "memory_limit": "4GB"
    }
  }'
